# learning_laravel

## Server

### Start PHP Server on c9

`php -S 0.0.0.0:8080 -t public/`

## Laravel

### Create new laravel project

`composer global require "laravel/installer=~1.1"`
`composer create-project laravel/laravel myapp`

### Generate Key

`php artisan key:generate`

### Serve Laravel App

`php artisan serve`
